## 重构 kvgui 为 Windows 和 macOS 双平台客户端
date: 2026-03-01

- 将原 `cmd/kvgui/` 拆分为 `cmd/kvgui/windows/` 和 `cmd/kvgui/macos/` 两个独立 Wails 子项目
- `windows/main.go` 使用 `windows.Options`，`macos/main.go` 使用 `mac.Options`（含 `TitleBarHiddenInset` 内嵌标题栏）
- 每个平台子目录包含独立的 `frontend/` 和 `app.go` 副本，满足 Go `//go:embed` 不能引用包目录外文件的约束
- 新增 `cmd/kvgui/macos/build/darwin/Info.plist`，包含 `CFBundleExecutable` 等 macOS 应用包必要元数据（修复"已损坏"提示）
- Makefile 新增 `gui-windows` 和 `gui-macos` 构建目标，`gui` 作为 `gui-windows` 的别名保持兼容；`gui-macos` 使用 `~/go/bin/wails` 完整路径
- 更新 CLAUDE.md、specs/design.md、specs/spec.md 文档，反映新的双平台目录结构和构建命令
