## 新增 kvgui 桌面 GUI 客户端
date: 2026-02-23

- 新增 `cmd/kvgui` 模块，基于 Wails v2 + WebView2 实现 Windows 桌面 GUI 客户端，编译为单个 `.exe`
- Go 后端 `app.go` 封装 `pkg/client` SDK，暴露 Connect / Disconnect / SetKey / GetKey / DeleteKey / ExistsKey / GetTTL / GetStats / TriggerSnapshot 方法，统一返回 `Result{success, message, data}` 结构
- 前端采用纯 HTML/CSS/JS（无 npm 构建），通过 `window.go.main.App.*` 全局调用 Go 方法，规避 ES module 静默失败导致按钮无响应的问题
- UI 包含：连接栏（Host/Port + 状态指示器）、操作面板（Set/Get/Del/Exists/TTL Tab 切换）、仿终端输出日志、统计仪表盘（5 张卡片 + 请求明细）、快照触发按钮
